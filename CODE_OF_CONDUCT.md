# Code of Conduct

AiPanel 社区希望保持开放、专业、尊重事实的协作氛围。

## 我们鼓励

- 尊重不同经验水平的参与者；
- 清晰描述问题和证据；
- 聚焦技术事实和可复现行为；
- 对安全问题保持谨慎；
- 主动脱敏日志、密钥和服务器信息。

## 我们不接受

- 人身攻击、骚扰、歧视或威胁；
- 发布他人的隐私、密钥、服务器凭据；
- 在公开渠道传播可直接利用的漏洞细节；
- 恶意提交破坏性代码或后门；
- 持续偏离主题、刷屏或干扰维护工作。

## 安全问题

如果你发现安全漏洞，请不要直接发公开 issue。请按照 [SECURITY.md](./SECURITY.md) 的方式上报。

## 处理方式

维护者可以对违反准则的内容进行编辑、隐藏、关闭或删除，也可以限制相关账号继续参与项目讨论。
